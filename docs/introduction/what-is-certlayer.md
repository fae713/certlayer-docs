# What is CertLayer?

CERTLAYER is a GenLayer-powered accountability, reliability, and security platform for Web3 infrastructure.

It enables protocols to operate with verifiable guarantees by combining transparent incident workflows, automated enforcement mechanisms, and integrated security monitoring.

CERTLAYER is not a single dashboard or monitoring tool — it is a multi-surface platform combining operational enforcement, public accountability, and coordinated security monitoring.

## The System Includes

- **Public Reputation Explorer** — transparent protocol reliability visibility
- **Private Protocol Dashboard** — for managing incidents, commitments, and compensation workflows
- **Security Monitoring** — through HackDetection, including transaction analysis and exploit risk scoring
- **Admin and Security Control Surfaces** — for incident enforcement and security coordination
- **API Services** — enforcing authentication, ownership boundaries, and operational workflows
- **GenLayer Contracts** — anchoring reliability events and security actions on-chain

CERTLAYER is designed so each protocol operates within its own controlled environment while global administrative controls maintain network-level security oversight.
