# Use Cases

## DeFi Protocol Downtime Protection

Protocols can guarantee uptime to users.

If the protocol experiences downtime beyond an acceptable threshold, users are compensated automatically.

## Oracle Failure Compensation

DeFi protocols rely heavily on oracle price feeds.

If an oracle stops updating data, CertLayer can trigger automatic protection mechanisms.

## Bridge Security Monitoring

Cross-chain bridges are common targets for exploits.

CertLayer can monitor bridge infrastructure and trigger compensation if a failure occurs.

## Infrastructure Reliability Proof

Protocols can demonstrate infrastructure reliability by publishing monitoring results verified by CertLayer.
