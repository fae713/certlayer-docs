# CERTLAYER Roadmap

## Phase 1: Make it Stable

- Save data in database (not memory only)
- Improve auth, permissions, and validation
- Add logs and monitoring

## Phase 2: Reliability Core

- Complete incident flow:
  - Create
  - Challenge
  - Dispute
  - Finalize
  - Payout
- Improve worker monitoring
- Make protocol dashboard fully production-ready

## Phase 3: Protocol Economics

- Add clear pricing tiers
- Add minimum pool rules
- Add payout and fee calculations
- Show pool health and billing in dashboard

## Phase 4: HackDetection Integration

- Improve security analysis and risk scoring
- Add emergency pause workflows
- Add protocol integration guide (`should_pause_protocol` checks)

## Phase 5: Public Explorer + API Product

- Improve public reputation explorer
- Launch API key plans and usage tracking
- Add webhook alerts for incidents and score changes

## Phase 6: Production Launch

- Security audit
- Performance and reliability hardening
- Launch with onboarding docs and support process
